# GUI Image Editor App 🖼️

## 📌 Description
A simple GUI-based image editor built using Python, OpenCV, and Tkinter.

## 🚀 Features
- Load Image
- Grayscale Conversion
- Blur Effect
- Edge Detection
- Save Image

## 🛠️ Technologies
- Python
- OpenCV
- Tkinter
- Pillow

## ▶️ How to Run
pip install -r requirements.txt
python main.py
